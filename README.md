# Frontsite
Create a site for your app.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'frontsite'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install frontsite
```

## Usage

Run the generator:
```bash
$ bin/rails g frontsite:install
```
